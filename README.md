# SQL_Retail_Project
# Retail Sales SQL Analysis Project

## Project Overview
This is my first end-to-end SQL project where I analysed a retail sales dataset to answer real business questions. The project covers database setup, data cleaning, data exploration, and business analysis using MySQL.

---

## Database Setup

- **Database:** `sql_project_p2`

| Column | Data Type | Description |

## Business Questions & SQL Queries

### Q.1 Retrieve all sales made on '2022-11-05'
```sql
SELECT *
FROM retail_sales
WHERE sale_date = '2022-11-05';
```

### Q.2 Clothing transactions with quantity >= 4 in November 2022
```sql
SELECT * 
FROM retail_sales
WHERE category = 'Clothing'
AND DATE_FORMAT(sale_date, '%Y-%m') = '2022-11'
AND quantity >= 4;
```

### Q.3 Total sales for each category
```sql
SELECT category,
    SUM(total_sale) AS sum_of_sale
FROM retail_sales
GROUP BY category;
```

### Q.4 Average age of Beauty category customers
```sql
SELECT
    ROUND(AVG(age), 2) AS avg_age
FROM retail_sales
WHERE category = 'Beauty';
```

### Q.5 All transactions where total sale is greater than 1000
```sql
SELECT * 
FROM retail_sales
WHERE total_sale > 1000;
```

### Q.6 Total transactions by gender in each category
```sql
SELECT category, gender, 
    COUNT(*) AS total_transaction
FROM retail_sales
GROUP BY gender, category
ORDER BY category;
```

### Q.7 Best selling month in each year by average sale
```sql
SELECT * FROM (
    SELECT
        YEAR(sale_date) AS year,
        MONTH(sale_date) AS month,
        AVG(total_sale),
        RANK() OVER(PARTITION BY YEAR(sale_date) ORDER BY AVG(total_sale) DESC) AS ranking
    FROM retail_sales
    GROUP BY year, month
) AS t1
WHERE ranking = 1;
```

### Q.8 Top 5 customers by total sales
```sql
SELECT customer_id,
    SUM(total_sale) AS total_sales
FROM retail_sales
GROUP BY customer_id
ORDER BY total_sales DESC
LIMIT 5;
```

### Q.9 Number of unique customers per category
```sql
SELECT category,
    COUNT(DISTINCT customer_id)
FROM retail_sales
GROUP BY category;
```

### Q.10 Total revenue generated by each gender
```sql
SELECT gender,
    SUM(total_sale) AS total_revenue
FROM retail_sales
GROUP BY gender;
```

### Q.11 Best selling category by revenue in each month
```sql
SELECT * FROM (
    SELECT 
        DATE_FORMAT(sale_date, '%Y-%m') AS month,
        category,
        SUM(total_sale) AS total_revenue,
        RANK() OVER(PARTITION BY DATE_FORMAT(sale_date, '%Y-%m') 
        ORDER BY SUM(total_sale) DESC) AS ranking
    FROM retail_sales
    GROUP BY DATE_FORMAT(sale_date, '%Y-%m'), category
) AS monthly_sales
WHERE ranking = 1;
```

### Q.12 Customers with more than 3 transactions
```sql
SELECT customer_id, COUNT(transactions_id) AS total_count
FROM retail_sales
GROUP BY customer_id
HAVING COUNT(transactions_id) > 3;
```

### Q.13 Percentage contribution of each category to total revenue
```sql
SELECT category,
    SUM(total_sale) AS sum_total,
    SUM(total_sale) / (SELECT SUM(total_sale) FROM retail_sales) * 100 AS percentage
FROM retail_sales
GROUP BY category;
```

### Q.14 Most recent transaction for each customer
```sql
SELECT 
    customer_id,
    MAX(sale_date) AS most_recent_transaction
FROM retail_sales
GROUP BY customer_id;
```

### Q.15 Customers who purchased from more than one category
```sql
SELECT customer_id,
    COUNT(DISTINCT category) AS category_count
FROM retail_sales
GROUP BY customer_id
HAVING COUNT(DISTINCT category) > 1;
```

### Q.16 Month with highest transactions for each category
```sql
SELECT * FROM (
    SELECT 
        DATE_FORMAT(sale_date, '%Y-%m') AS month,
        category,
        COUNT(transactions_id) AS total_transactions,
        RANK() OVER(PARTITION BY category ORDER BY COUNT(transactions_id) DESC) AS ranking
    FROM retail_sales
    GROUP BY DATE_FORMAT(sale_date, '%Y-%m'), category
) AS highest_transactions
WHERE ranking = 1;
```

### Q.17 Rank customers by total spending
```sql
SELECT customer_id,
    SUM(total_sale) AS total_spending,
    RANK() OVER(ORDER BY SUM(total_sale) DESC) AS ranking
FROM retail_sales
GROUP BY customer_id;
```

### Q.18 Running total of sales ordered by date
```sql
SELECT sale_date,
    SUM(total_sale) AS daily_sale,
    SUM(SUM(total_sale)) OVER(ORDER BY sale_date) AS running_total
FROM retail_sales
GROUP BY sale_date;
```

### Q.19 Average transaction value per age group
```sql
SELECT 
    CASE
        WHEN age BETWEEN 18 AND 25 THEN '18-25'
        WHEN age BETWEEN 26 AND 35 THEN '26-35'
        WHEN age BETWEEN 36 AND 50 THEN '36-50'
        ELSE '51+'
    END AS age_group,
    ROUND(AVG(total_sale), 2) AS avg_transaction
FROM retail_sales
GROUP BY age_group;
```

### Q.20 Customers with no purchase in the last 3 months of the dataset
```sql
SELECT DISTINCT customer_id
FROM retail_sales
WHERE customer_id NOT IN (
    SELECT DISTINCT customer_id
    FROM retail_sales
    WHERE sale_date >= DATE_SUB(
        (SELECT MAX(sale_date) FROM retail_sales), 
        INTERVAL 3 MONTH
    )
);
```
